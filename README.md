# signal-to-noise.co

Personal site + insights blog. Astro · Tailwind · MDX.

See [`signal2noise.md`](./signal2noise.md) for the full project brief.

## Develop

```bash
npm install
npm run dev          # http://localhost:4321
npm run build        # static output → ./dist
npm run preview      # serve the production build locally
```

## Writing

- Insights: `src/content/insights/*.mdx`
- Case studies: `src/content/case-studies/*.mdx`

Frontmatter schemas are defined in `src/content.config.ts`.
